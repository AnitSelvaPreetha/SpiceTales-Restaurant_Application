
Restaurant Management System


Overview
     This Restaurant Management System is designed to streamline restaurant operations for both administrators and customers.
It provides functionalities for managing dishes, reservations, table allocations, customer orders, feedback, and more.

User Stories
Admin Stories
As an admin, I want to add new dishes so that I can update the menu.
As an admin, I want to approve reservations so that customers can dine at scheduled times.
As an admin, I want to allocate tables when approving reservations so that seating is managed efficiently.

Customer Stories
As a customer, I want to register so that I can place orders.
As a customer, I want to add dishes to my cart so that I can customize my meal.
As a customer, I want to view my past orders so that I can reorder my favourite dishes.
As a customer, I want to submit feedback so that I can share my dining experience.
